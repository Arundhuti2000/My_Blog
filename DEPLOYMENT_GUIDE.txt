============================

STEP 1: Prerequisites
- Install Node.js (https://nodejs.org)
- Install Firebase CLI: npm install -g firebase-tools

STEP 2: Firebase Project Setup
1. Go to https://console.firebase.google.com
2. Create a new project or select existing one
3. Enable Firebase Hosting in the console

STEP 3: Local Setup
1. Extract the downloaded files
2. Open terminal in the project folder
3. Run: npm install
4. Run: firebase login
5. Run: firebase init hosting
   - Select your Firebase project
   - Set public directory to: out
   - Configure as single-page app: Yes
   - Set up automatic builds: No

STEP 4: Update Configuration
1. Open .firebaserc file
2. Replace "your-project-id" with your actual Firebase project ID

STEP 5: Deploy
1. Run: npm run build
2. Run: firebase deploy

STEP 6: Access Your Blog
Your blog will be available at:
https://your-project-id.web.app

TROUBLESHOOTING:
- If build fails, check Node.js version (needs 18+)
- If deploy fails, verify Firebase project ID in .firebaserc
- For permission issues, run: firebase login --reauth

CUSTOMIZATION:
- Edit content in app/ folder
- Replace images in public/ folder
- Update colors in tailwind.config.js
- Modify styles in app/globals.css
